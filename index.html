Poker.js — Growl-esque notifications for your web app

Examples

  • Default notification
  • With a long message
  • With a timeout of 1s before fade
  • With basic markup or an image
  • Without the “×”
  • Aligned to the left
  • Aligned to the bottom left
  • Aligned to the bottom right

Download

You can find the latest release on GitHub.

Installation

Make sure you have jQuery loaded, or load it from a public CDN:

<script src="https://ajax.googleapis.com/ajax/libs/jquery/2.1.4/jquery.min.js"></script>

Grab the following files from the repo and put them in your page <head>:

<script src="poker.js"></script>
<link href="poker.css" rel="stylesheet" type="text/css" />

Signature

Poker.poke(message, options = {});

Usage

Poker.poke('This is a test of a default poke'); Poker.poke('This is a test of a quick poke', { timeout: 1000 });

Configuration

These are all the options you can override in the options parameter:

{
  close_button: true,
  container: {
    css: {
      maxWidth: '400px',
      padding: '0 12px',
      position: 'fixed',
      zIndex: '9999'
    }
  },
  timeout: 5000,
  x: 'right',
  y: 'top'
}
